카메라 촬영
모발을 직접 촬영해서 색상을 자동으로 감지해요.
촬영 후 슬라이더로 보정 가능해요.
✏️
수동 분석
카메라 없이 색상을 직접 선택해요.
색상 프리셋 + 슬라이더로 정밀하게 설정 가능해요.
모발로 화면을 꽉 채워주세요 · 배경 노출 시 오류 가능
📋 촬영 전 확인
💡직사광선·강한 조명 피하기 — 자연광이나 일반 실내조명에서 촬영
🎯모발로 가이드 박스를 꽉 채워주세요 — 배경(피부·옷)이 조금이라도 찍히면 오류 가능
⚠️형광등·할로겐·전구색 조명에서는 실제 색감과 다르게 인식될 수 있어요
